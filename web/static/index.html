What this project is

WargamingLLM is a multi-agent simulation of the fights that will decide how farmed animals are treated. Each run casts language models as real stakeholder types — vertically integrated producers, contract growers, corporate campaigners, alternative-protein companies, regulators, ESG investors, retail buyers, and industry defense coalitions — each with a researched objective describing what that actor actually wants and how it actually behaves. A neutral judge model resolves their moves turn by turn into realistic outcomes, then writes an after-action report on where things ended up.

No single run predicts anything. The point is repetition: run the same scenario many times, vary the models and lineups, and look for the patterns that keep showing up — which levers reliably move welfare outcomes, which commitments reliably collapse, which coalitions form again and again. That aggregate picture is what this page will present.

The scenarios under study

Six starting situations are being explored, each built from real dynamics in animal agriculture. Findings for each will appear here.

Cage-Free Reckoning

A retailer three months from a public 100% cage-free deadline it won't hit, an advocacy scorecard about to call it a laggard, and an avian-flu supply shock hitting everyone at once. What happens to corporate welfare pledges under pressure?

What to look for when the analysis lands

  • Which mechanisms actually moved welfare outcomes across runs — public scorecards, supplier contracts, shareholder pressure, regulation, market alternatives — and which just made noise.
  • The failure modes: where welfare commitments collapsed, and what conditions made collapse cheap for the actor abandoning them.
  • Recurring alliances and betrayals — especially the unstable ones, like growers and welfare advocates, or investors and campaigners.
  • How outside shocks (disease outbreaks, investigations, technology shifts) redistributed leverage between actors.
  • Whether different models playing the same role behave differently — and what that says about using LLMs for this kind of foresight at all.

Methodology note: every claim on this page will be traceable to specific runs viewable in the Results tab. Simulations are exploratory foresight, not prediction — they map the space of plausible dynamics, not the future.

About this project

WargamingLLM is a configurable multi-agent policy wargame: pick stakeholder agents from a fixed roster of researched roles (each with a detailed objective you can view), choose which model plays each of them, choose a model for the neutral judge (whose grounding context is built in), and write your own starting scenario and turn count. Each turn, every agent privately decides what it does given the current situation; the judge then resolves all of those actions into a realistic outcome, which becomes the next turn's starting situation. The result is a turn-by-turn transcript of how different interests might actually behave and collide under a given scenario.

Model calls run through OpenRouter (a single API key drives any mix of models across agents and the judge), or — when running this site locally — through the Claude Code CLI on the host machine, with no API key at all: calls are billed to the Claude account the CLI is logged into.
