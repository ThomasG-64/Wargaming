About this project

WargamingLLM is a configurable multi-agent policy wargame: define any number of stakeholder agents (a name, an objective, and which model plays them), a neutral judge (a model plus background context to ground its rulings), a starting scenario, and a turn count. Each turn, every agent privately decides what it does given the current situation; the judge then resolves all of those actions into a realistic outcome, which becomes the next turn's starting situation. The result is a turn-by-turn transcript of how different interests might actually behave and collide under a given scenario.

All model calls run through OpenRouter, so a single API key can drive any mix of models across agents and the judge.
